Description détaillée

Cette classe définit un Creneau à partir de son jour et le tableau de ses disponibilités associé.

Au-delà de définir un Creneau, cette classe permet aussi de désigner l'Etudiant associé au Creneau.

L'Etudiant correspondant au Creneau est défini par un pointeur.

Avertissement
Un Creneau a 1 seul Etudiant associé.
