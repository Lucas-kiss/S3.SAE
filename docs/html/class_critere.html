Description détaillée

Cette classe définit les Critere associés à l'Offre.

Cette classe comprend le nbMinHeureEtudJour (nb min d'heures recherché par étudiant par jour), si heureRepartieJour ou non, le nbLinEtudJour (nb min d'étudiant recherché par jour), le nbMinEtudTotal (nb min d'étudiant recherché sur toute la semaine) et l'Offre associée.
